Keyboard Shortcuts
------------------
Navigation:
  Tab / → / l     Next tab
  Shift+Tab / ← / h   Previous tab
  1-6             Jump to tab

Actions:
  d               Load data
  p               Run prediction
  a               Detect anomalies
  s               Generate signals
  e               Export results
  c               Configure
  r               Refresh
  q               Quit

Server:
  v               Start/view server
  Ctrl+C          Stop server
