### Before Asking

1. Check the [documentation](doc/overview.md)
2. Search [existing issues](https://github.com/rustful-ts/rustful-ts/issues)
3. Review [closed issues](https://github.com/rustful-ts/rustful-ts/issues?q=is%3Aissue+is%3Aclosed)

### Creating an Issue

When reporting bugs, include:

1. **Version**: rustful-ts version and runtime (Node.js/Bun/Browser)
2. **Environment**: OS, Node.js version
3. **Steps to Reproduce**: Minimal code example
4. **Expected Behavior**: What should happen
5. **Actual Behavior**: What actually happens
6. **Error Messages**: Full error output

### Feature Requests

For feature requests, include:

1. **Use Case**: What problem does this solve?
2. **Proposed Solution**: How should it work?
3. **Alternatives**: Other approaches considered
4. **Examples**: Code examples or references

## Response Times

- **Bug Reports**: Triaged within 7 days
- **Feature Requests**: Reviewed within 14 days
